# Per Aspera Mod Registry

Source officielle des mods installables via le **PerAspera Mod Launcher**.

- [`registry.json`](registry.json) — la liste curée des mods (lue par le launcher)
- [`CONVENTIONS.md`](CONVENTIONS.md) — format de packaging et workflow de publication

## Ajouter votre mod

1. Packagez vos releases au format standard (voir CONVENTIONS.md)
2. Ouvrez une PR ajoutant votre entrée dans `registry.json`

## Écosystème

| Brique | Repo |
|--------|------|
| SDK de modding | [PerAspera-SDK](https://github.com/PerAsperaMods/PerAspera-SDK) |
| Workflow de release réutilisable | [.github](https://github.com/PerAsperaMods/.github) |
| Launcher (à venir) | PerAspera-Launcher |
